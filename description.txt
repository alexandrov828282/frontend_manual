поик:

начинаем вбивать
по мере вбития в выпадающей строке вываливаются совпадения
можно выбрать в совпадениях нужное если есть
по нажатию enter на страницу помещается выдача подходящих по тегам статей
по нажатию ctrl+P на странице появляются кнопки разработчика - "создать", "редктировать".

страница создания статьи:

заполняется страница-форма:

название (в процессе заполнения формируется транкрипт название будущего файла, который нужно создать вручную)
синтаксис
определение
примеры
подробности
особенности
теги (теги можно добавлять и редктировать на месте (правка базы данных тегов))

по каждому пункту галочка, помещаеть раздел на страницу статьи или нет

в пункты примеры, особенности, подробности есть возможность вставлять фрагменты кода (без заморочек)

по нажатию кнопки "готово" формируется html код страницы в отдельный существующий файл, в файле в комментарии помещается название файла, еоторый нужно создать и наполнить сформированным содержимым, добавляется запись в базу
